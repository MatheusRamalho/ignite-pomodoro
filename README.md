## 💡 Projeto

Projeto da rocketseat proposto no curso de React da Rocktseat. Um Pomodoro onde você seta um tempo e da um nome para esse timer.
Tambem possui uma página de histórico.

<br />

## ⭐ Stack

- React
- Next
- Typescript
- Tailwind
- Git
- Github
- Vercel
